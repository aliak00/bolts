    Alias CommonTypeOfRanges

    Finds the CommonType of a list of ranges

    alias CommonTypeOfRanges(Rs...) = CommonType!(staticMap!(ElementType,Rs));

    Example

    auto a = [1, 2];
auto b = [1.0, 2.0];
static assert(is(CommonTypeOfRanges!(typeof(a), typeof(b)) == double));
